# Prague Parking 2.0 - G Grade

Detta är min lösning på Prague Parking 2.0.

## Hur man kör programmet

1.  Öppna `PragueParkingSolution.sln` i Visual Studio.
2.  Se till att projektet `PragueParking.App` är startprojektet.
3.  Bygg lösningen (`Ctrl+Shift+B`).
4.  Kör programmet (`F5` eller `Ctrl+F5`).
5.  Följ menyinstruktionerna i konsolfönstret.

## Viktig information

*   Programmet använder `Spectre.Console` för gränssnittet.
*   Konfiguration läses in från `config.json` vid start. Om filen inte finns används standardvärden.
